2.2 When we push code to root branch, by default it has name as "master" which sound really discriminating so generally developer rename it as "main".
    > master branch must be active, then:
		$ git branch -M main
	> Note: you can given any name other than "main" but "main" is prefered as standared.

	#Summary:
		> git branch -M main
		> git branch -M amir_root_branch
